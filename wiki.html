👋기린위키

기린월드에서 재밌는 시간 보내세요!

악어의 놀이터 모티브 서버 기린월드입니다.

  • 서버 주소 : giraffemc.com

  • 서버 버전 : 1.20.1

  • 서버 난이도 : 어려움

  • 약탈 금지

  • 디스코드

  • 마인리스트

서버의 공지사항 및 패치내역은 디스코드에서 안내됩니다.

서버 소개

악어의 놀이터를 모티브로 하여 제작된 기린월드 입니다! 서버의 기반은 악어의 놀이터 기반으로 하여, 유저간의 경제 활동 활성화를 중요하게 생각한 반야생 서버입니다. 악어의 놀이터에서 보지 못한 다양한 아이템 레벨 업 및 강화 방식을 꾸준히 추가하여 재미를 더할 예정입니다. 현재도 자체 제작 플러그인들과 외부 플러그인들을 더하여 많은 컨텐츠를 열심히 제작 중입니다!

  • 처음 입장시 튜토리얼이 진행됩니다.

튜토리얼은 모든 직업을 직접 체험하실 수 있습니다.

튜토리얼 완료 시 전직이 무료로 제공 됩니다.

튜토리얼 완료 시 기본 아이템과 함께 소량의 보상금이 지급됩니다.

  • /길라잡이를 따라가며 초반 컨텐츠를 익혀보세요.

  • SHIFT + F를 이용하여 월드 이동이 가능합니다.

  • 신규 플레이어는 24시간 동안 인벤세이브가 활성화됩니다. (서버 최초 접속시간 기준 24시간)

  • 지옥 월드는 초급 직업부터, 엔더 월드는 고급 직업부터 입장하실 수 있습니다.

  • 초반 컨텐츠 (추천)

일일 퀘스트를 따라가며 약간의 돈과 퀘스트 포인트를 확보하고, 광물을 캐거나 바닐라 작물을 농사하여 NPC에게 판매하여 돈을 확보하여 전직을 하면 더 빠른 속도로 수입이 늘어납니다.

  • 자바 에디션 1.20.1 버전으로 마인크래프트를 실행합니다.

  • 메인메뉴에서 멀티플레이로 입장합니다.

  • 하단의 서버 추가 버튼을 클릭합니다.

  • 서버 주소에는 giraffemc.com를 입력하고, 서버 리소스팩은 사용으로 변경해줍니다.

1. 패브릭(fabric) 설치

아래 설치 링크를 클릭 하여 구글 드라이브에서 다운로드 하세요.

꼭 설치 전에 1.20.1 마인크래프트를 실행 후에 설치를 해야 정상적으로 설치됩니다.

설치 시 반드시 1.20.1버전으로 설치해 주세요.

패브릭 설치 링크:

https://drive.google.com/file/d/1yw7dDIFF5CYDDDAZYdiU9eCfSOf7Ykto/view?usp=sharing

2. 모드 적용

  1. 윈도우 + R 키를 눌러 나타난 실행창에 %appdata%를 입력 후 확인 클릭

  2. 나타난 폴더에서 .minecraft 폴더를 찾아서 열기 .minecraft 폴더 내의 mods 폴더에 모드 파일을 옮겨 넣기 (mods 폴더가없다면 마우스 우클릭 > 새풀더 > 이름을 mods로 변경해 새롭게 만들어주세요.)

  3. 마인크래프트 런처를 실행하고 실행 버튼 좌측에 버전을 아래와 같이 변경하고 실행 하세요.

버전에 패브 프로필이 나타나지않는다면,마인크래프트 런처 상단에 설치 설정 클릭 > 새 설치 설정 > 버전에서fabric-loader-버전을 찾아 선택 후 만들기다시 상단의 플레이를 눌러 돌아와서 플레이 왼쪽에 버전을 새로 만든 버전으로 선택 후 플레이

3. 추천 모드

위 구글 드라이브 파일에 모두 첨부 되어 있습니다.

  1. 한글 채팅 모드 (caramelChat)

    한/영키를 누르지 않고도 한글 채팅 후 움직임이 가능한 모드입니다.

  2. 조합법 확인 모드 (jei) 인게임 내 조합법을 쉽게 확인할 수 있도록 도와주는 모드입니다.

  3. 미니맵 모드 (journeymap) 내 위치 등을 편하게 확인하고, 사망 시 사망 장소도 자동으로 표기해 주는 모드입니다.

  4. 소듐(sodium) + 아이리스(iris) 쉐이더 팩을 사용 할 수 있도록 해주는 모드
